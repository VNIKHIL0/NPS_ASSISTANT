# NPS Assistant - AI Pension Planner

This is a [Next.js](https://nextjs.org) project designed to help users plan their retirement with the National Pension System (NPS) using advanced AI.

## Features
- **AI Pension Planner**: Personalized retirement roadmap generation.
- **Multilingual Support**: Supports all 22 official Indian languages.
- **Pension Calculator**: Interactive tool to visualize future wealth.
- **Financial Literacy**: Comprehensive guides on NPS.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
